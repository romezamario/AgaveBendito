"Woo Shortcodes Kit" now called WSHK is a easy Functions & Shortcodes kit which can be displayed on any page or post to build your own my account page and customize the shop page, order emails, the add to cart button and much more!. Enjoy customizing easilly your WooCommerce's shop with more than 60 shortcodes & functions. This plugin not work alone, you need install WooCommerce before.


Now you can adapt you store to the new GDPR law fully and easilly!

== Description ==

Customize your WooCommerce store with more than 60 functions and shortcodes, ranging from the important points of your store such as: Access, page my account, shop page, thank you pages among others, to additional content to reinforce as content restriction, adaptation to the new law of data protection, accountants, security, advanced user data, messages according to the number of orders among many more advantages.

"Woo Shortcodes Kit" does not work alone. It is originally created for [WooCommerce](https://wordpress.org/plugins/woocommerce/), so before install "Woo Shortcodes Kit" you must have need to install WooCommerce on your website.

**Tested & Running with WorPress v5.2.2 | WooCommerce v3.7.0 (27/09/19)

== Installation ==

In most cases you can install automatically from WordPress.

However, if you install this manually, follow these steps:

1. Upload the entire `woo shortcodes kit` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Check the Admin menu/ WooCommerce / WSHK

= What can do each Shortcode? =

Now the shortcodes are ordered by 3 different categories: BUILD MY ACCOUNT | COUNTERS | ADDONS. You can use the new Shortcodes Filter to display only the shortcodes of the clicked categorie.

**BUILD MY ACCOUNT**
This categorie display all the shortcodes made to build you custom my account page.

1) **Orders List**: with the **NEW** shortcode [woo_myorders] you can show a list with the Orders in your custom my account page and can view the orders details just below. Will can the orders details doing just a click in the new close button added.

If a user dont have made any order yet, then will appear the WooCommerce's default message "You dont have any order yet" and the button "GO TO SHOP".

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_myorders]


2) **Downloads List**: with the **NEW** shortcode [woo_mydownloads] you can show a list with the Downloads in your custom my account page and download with just a clic the file included in the product.

When a user buy a digital product and it have a file to download, appear a table list, so if make click in the product name the user go to product page and if make click in the file link the user start to download. If you have configurate in product settings that the files have a expire date or limited number of downloads this details will appear too in the downloads table.

If a user dont have made any order yet, then will appear the WooCommerce's default message "You dont have any download yet" and the button "GO TO SHOP"

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_mydownloads]


3) **My Addresses** with the **NEW** shortcode [woo_myaddress] you can show the user's billing and shipping addresses just like how do WooCommerce by default. The user can edit his addresses with just a click and see how appear the form edit address just below. After save the changes, the form edit will dissappear and just will see the new saved addresses.

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_myaddress]

4) **Payment Methods** with the **NEW** shortcode [woo_mypayment] you can show the user's saved cards, but will displayed too a button to add new cards. This function work like than WooCommerce's function so you need have Stripe or similar how installed payment methods.

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_mypayment]

5)**Edit Account** with the **NEW** shortcode [woo_myedit_account] you can show to the users the edit account form. It work how WooCommerce's function.

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_myedit_account]

6)**Dashboard** with the **NEW** shortcode [woo_mydashboard] you can show the WooCommerce's dashboard, this function have hide the default WooCommerce's Dashboard message. But sometimes the plugins display info in the Dashboard, so if you need display it just need use this shortcode. In the next version will include the option to show/hide the WooCommerce's Dashboard by default message.

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_mydashboard]

7)**Logout button** with the shortcode [woo_logout_button] you can show a custom link with your own style to let close the session to the logged in users. From function advanced options you can build your own button style and add too a custom redirection to send the users after logout. Just need add your custom page slug in the available field.

This shortcode let you write a custom button text so you choose the language.

VERY IMPORTANT!! : Normally the button is added into the custom my account page, but can be included everywhere, just using the shortcode [woo_logout_button]


8)**Login & Register Form** with the **NEW** shortcode [woo_login_form] you can show the WooCommerce's Login and Register form, this function work how the original form that WooCommerce provide, so the Lost password link and reset password will work perfectly. 

You can control where you want redirect to the user after the login, writing your custom page slug in the available field that you see when expand the box to see the function advanced options.

This shortcode take the WooCommerce's language so you dont need translate nothing.

VERY IMPORTANT!! : You need set how the WooCommerce's my account, the page that you have included the [woo_login_form]

**COUNTERS**
This categorie display all the shortcodes made to help you to count actions or products in your web.

1) **Show Global sales**, with this simple shortcode [woo_global_sales] you can put everywhere (pages,posts, portfolio....) the number of the total sales that you have made in your store for the users. 



2) **Show the total products** that you have in your shop, for example if you have 20 products in total, and use the shortcode [woo_total_product_count] then will appear the number 20 where you want (pages,portfolio,product page, shop page...)

But you can exclude any category too, for example if you have 3 categories: PLUGINS, THEMES, MUSIC and wont count the MUSIC category in the total products count, just need add: cat_id= "HERE WRITE YOUR CATEGORY ID NUMBER"

so the shortcode will be [woo_total_product_count cat_id="3"]

The number 3 is the ID number of the MUSIC category.

3) **Total products bought by user counter**: with the shortcode [woo_total_bought_products] you will get the number total in any page, post, portfolio...

4) **Total orders made by user counter**: with the shortcode [woo_customer_total_orders] you will get the number total in any page, post, portfolio...
	

5) **Total reviews made by user counter**: with the shortcode [woo_total_count_reviews] you will get the number total in any page, post, portfolio...

**ADDONS**
This categorie display all the shortcodes made to show extra information about the user or hide...

1) **Bought products**: with the shortcode [woo_bought_products] you can display a loop with the products that a user has bought. 


2) **Gravatar image**: with the shortcode [woo_gravatar_image] you can display the user's Gravatar image in any page, post, portfolio...


3) **Review by user**: with the shortcode [woo_review_products] you will get all the reviews in any page, post, portfolio...

Now the review, display the product title linked to the buyed product.

4)**Display all the user reviews everywhere**: with the shortcode [] you will show all the reviews in any page, post, porftolio...

Can manage the styles, the number of reviews that you want display and in how many columns.

5) **Username**: with the shortcode [woo_user_name] you will get the username in any page, post, portfolio...

**Now** display the name of the client, in the next version will be choose between username, name and others...
	

6) **Message by orders**: with the shortcode [woo_message] you will set a custom message to the user if have made X orders, can display in any page, post, portfolio...
	

7) **Restrict content for non logged in users**: with the nested shortcode [wshk]Put here your content[/wshk] you will restrict content to the non logged in user's in any page, post, portfolio...

Can restrict simple text, HTML, shortcodes...


8) **Hide content for logged in users**: with the nested shortcode [off]Put here your content[/off] you will hide content to the logged in user's in any page, post, portfolio...

Can hide simple text, HTML, shortcodes...

9) **Display the user IP**: with the shortcode [woo_display_ip] you will show where you want the current user IP address, not need activation on settings page, just copy & paste.

10) **Display the user Name and surname**: with the shortcode [woo_display_nsurname] you will show where you want the current user Name and surname, not need activation on settings page, just copy & paste.

11) **Display the user email**: with the shortcode [woo_display_email] you will show where you want the current user email, not need activation on settings page, just copy & paste.


Remember that some Shortcodes need activate his function from settings to work.
